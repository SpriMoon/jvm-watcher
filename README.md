# JVM Remote Watcher
### 配置运行jvm参数
```
-Djava.library.path=/jvm-watcher/sigar-bin
```
### JMX RMI URI格式
```
service:jmx:rmi:///jndi/rmi://127.0.0.1:8899/jmxrmi
```
